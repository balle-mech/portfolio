# Portfolio

個人ポートフォリオサイト - エンジニアとしてのスキル、経験、アウトプットを効果的に伝える。

## プロジェクト概要

Reactベースのモダンなポートフォリオサイト。エンジニアの名刺として、技術力と実績を視覚的に分かりやすく提示します。

### デザインコンセプト

- **ミニマル**: 余白を活かしたシンプルなデザイン
- **プロフェッショナル**: 落ち着いたダークカラーパレット
- **レスポンシブ**: モバイル/タブレット/デスクトップ全対応

### 主要機能

- **ヒーローセクション**: 名前・肩書きの表示
- **スキルスタック**: 技術スキルの視覚的な提示
- **アウトプット**: プロジェクト・制作物の紹介（準備中）
- **お問い合わせフォーム**: Formspree連携

## 技術スタック

- React 19.1.1
- TypeScript 5.9.2（strict mode）
- Vite 7.1.5
- Tailwind CSS 4.1.11
- Jest + React Testing Library

## 実装内容

- レスポンシブデザイン（モバイル対応）
- カスタムフック（`useHamburgerMenu`）によるUI状態管理
- Jest + React Testing Libraryによるテスト
  - ユーザーインタラクションテスト
  - DOM操作テスト
  - アクセシビリティ対応（Escapeキー等）
- ESLint + Prettierによるコード品質管理
